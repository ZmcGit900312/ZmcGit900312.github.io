3. 重置密码

3.1 打开另一个cmd窗口，输入mysql进入数据库

3.2 输入show databases;看到数据库说明进入成功

3.3 权限表放在mysql.user中

3.4 输入SELECT user, host, plugin, authentication_string from mysql.user;可以看见用户和加密后的密码

注意，plugin插件代表加密方式，mysql自带的是mysql_native_password

3.5 更改root密码

输入update user set password=password('123456') where user='root' and host='localhost';

3.6 再次查看密码

输入SELECT user, host, plugin, authentication_string from mysql.user;

4. 重启mysql

4.1 关闭cmd，并手动停止mysqld的运行

4.2 输入net start mysql重启mysql

4.3 输入mysql -u root -p 123456登录mysql
