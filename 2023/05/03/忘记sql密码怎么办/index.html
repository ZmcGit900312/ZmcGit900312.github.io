  1. 关闭sql
  2. 跳过权限表
  3. 重置密码
  4. 重启sql

重置密码

  1. 打开另一个cmd窗口，输入mysql进入数据库
  2. 输入show databases;看到数据库说明进入成功
  3. 权限表放在mysql.user中
  4. 输入SELECT user, host, plugin, authentication_string from mysql.user;可以看见用户和加密后的密码

    注意，plugin插件代表加密方式，mysql自带的是mysql_native_password

  5. 更改root密码，输入update user set password=password('123456') where user='root' and host='localhost';
  6. 再次查看密码，输入SELECT user, host, plugin, authentication_string from mysql.user;

重启mysql

  1. 关闭cmd，并手动停止mysqld的运行
  2. 输入net start mysql重启mysql
  3. 输入’mysql -u root -p 123456’登录mysql
