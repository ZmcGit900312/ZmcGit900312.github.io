10. 内置函数 Built-in Function

10.1 颜色函数 Color Functions

1o.1.1 RGB函数

red(color[,value])

返回第一个参数color的红色分量，或者给第二个可选参数value的红色分量进行赋值

green(color[,value])

与上一个函数类似，返回第一个参数color的绿色分量，或者给第二个可选参数value的绿色分量进行赋值

blue(color[, value])

返回第一个参数color的蓝色分量，或者给第二个可选参数value的蓝色分量进行赋值

alpha(color[, value])

返回第一个参数color的alpha值，或者给第二个可选参数value的alpha值进行赋值

dark(color)

检查color是否是黑色

light(color)

检查color是否是白色

rgba(color | r,g,b,a)

从r,g,b,a通道返回RGBA值

stylus同样支持#rgha和#rrggbbaa记法

rgb(color | r,g,b)

从r,g,b,a通道返回RGBA值或者转换为RGBA点

10.1.2 HSL函数

hue(color[,value])

返回第一个参数color的色调，或者给第二个可选参数value的色调进行赋值

saturation(color[,value])

返回第一个参数color的饱和度，或者给第二个可选参数value的饱和度进行赋值

lightness(color[,value])

返回第一个参数color的亮度，或者给第二个可选参数value的亮度进行赋值

hsla(color | h,s,l,a)

将给定的color转换为HSLA代码，或者 h,s,l,a 分量

hsl(color | h,s,l)

将给定的color转换为HSLA代码，或者 h,s,l,a 分量

1o.1.3 其他函数

blend(top[, bottom])

使用正常方式混合top和bottom，其中bottom是可选参数，默认为#fff

lighten(color, amount)

对color增强amount倍亮度，此函数与数值单位有关，比如

darken(color, amount)

对color降低amount倍亮度

desaturate(color, amount)

对color降低amount倍饱和度

saturate(color, amount)

对color提升amount倍饱和度

complement(color)

返回互补颜色，等效在色环上旋转180度

invert(color)

反转颜色，即反转r,g,b，透明度不变。

spin(color, amount)

对给定的color在色环上旋转amount角度

grayscale(color)

对给定的颜色进行灰阶转换，等效于desaturate(color, 100%)

mix(color1, color2[, amount])

混合两种颜色，amount为可选参数，默认为50%

tint(color, amount)

对给定颜色与白色进行混合

shade(color, amount)

对给定颜色与黑色进行混合

luminosity(color)

返回指定颜色的相对光照强度

contrast(top[, bottom])

返回top和bottom之间的对比度，第二个参数为可选参数，默认值为#fff。

transparentify(top[, bottom, alpha])

返回top或者与bottom的透明度，第二个参数默认为#fff，第三个参数为可选参数

10.2 路径函数 Path Function

basename[path[,ext]]

返回path指定的文件名，ext为可选参数，代表文件的扩展名

extname(path)

返回path的扩展名

pathjoin(...)

路径拼接

10.3 列表/哈希函数 List/Hash Functions

push(expr, args...)

将给定的args压入expr

pop(expr)

弹出expr的最后一个值

shift(expr)

从左边弹出一个值

unshift(expr, args...)

将给定args逆序排列在expr前

index(list, value)

返回list中value的索引值（从0开始）

key(pairs)

返回pairs的key

value(pairs)

返回pairs的value

list-separator(list)

返回list的分隔符

length([expr])

括号中的表达式expr可以看作元组，length()函数返回元组的长度

last(expr)

返回expr的最后一个值

10.4 单元函数 Unit Functions

typeof(node)

返回node的类型名，可以使用别名type-of和type

unit(val[, type])

返回一个val的单位，或者在不进行单位转换的条件，赋予value给定的type。

percentage(num)

将num转换为百分数

10.5 数学函数 Math Functions

abs(unit)

ceil(unit)

round(unit)

注：所有的舍入函数都可以给出可选参数precision，来指定精度

sin(angle)、cos(angle)、tan(angle)

angle同时支持角度和弧度制，使用角度时需要带上单位deg

min(a, b)、max(a, b)

even(unit)、odd(unit)

sum(nums)、avg(nums)

range(start, stop[, step])

返回start到stop（包括stop）之间的list，step为可选的间隔值，默认为1

base-convert(num, base, width)

将num转换为base进制，宽度为width值并返回

10.6 字符串函数 String Functions

match(pattern, string[, flags])

返回字符串中string的匹配表达式pattern(正则表达式)

replace(pattern, replacement, val)

将匹配表达式pattern中的匹配项替换为replacement，并返回val

join(delim, vals...)

在vals中插入间隔符delim

split(delim, val)

将val值使用delim来分割

substr(val, start, length)

从val中start位置裁剪length长度的字符，并返回裁剪值

slice(val, start[, end])

从val中提取指定片段

unquote(str | ident)

去除指定字符串的引号，并返回literal记法

convert(str)

与unquote()函数类似，只是把str转换为Stylus记法

s(fmt, ...)

s()函数与unquote()函数类似，它也返回Literal记法，但是它能接受类C语言sprint()函数式的字符串记法

10.7 工具函数 Utility Functions

called-from 属性

called-from属性返回函数调用链

current-media()

返回当前块的@media规则

+cache(keys...)

+cache函数是一个非常强大的内置函数，它允许你创建自己的可缓冲混合，这里”可缓冲混合“支持在第一次调用时将其内容应用到给定的选择器，在第二次调用时，会@extend第一次的选择器

+prefix-classes(prefix)

对当前类选择器或块进行前缀扩展

编译后

lookup(name)

寻找指定变量名，并返回变量值，若无法找到则返回null

编译后

define(name, expr[, global])

使用name来创建或者覆写现有变量名，global为可选参数，省略时表示作用域为局部

编译后

operate(op, left, right)

将运算符op放在left和right之间并返回值

selector()

返回当前选择器或者伪类

selector-exists(selector)

如果selector存在返回true

该方法不会考虑所处上下文进行相对区域判断

opposite-position(positions)

返回给定位置的相反位置

image-size(path)

返回指定path的图片width和height

embedurl(path[, encoding])

返回图片的路径url()，可选编码参数encoding默认值为base64，可指定为utf8

add-property(name, expr)

对给定的表达式expr，添加属性名name
