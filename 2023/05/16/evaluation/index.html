1. Introduction

The Evaluation Methodology is a tool to help one better understand the steps needed to do a quality evaluation.

By following these steps, a faculty member can learn what he or she needs to know to determine the level of quality of a performance, product or skill.

2. Methodology

The Evaluation Methodology consists of four main steps along with a set of sub-steps.

2.1 Define the parameters of the evaluation(定义评估参数)

The client:

  1. 明确评估需求
  2. 明确评估结果的作用
  3. 明确应该向评估对象报告的内容以及由谁做报告
  4. 明确评估者需要听取的汇报内容
  5. 选择要遵循的指导方针来实施评估
  1. determines the need for the evaluation.
  2. determines the use for the results of the evaluation.
  3. determines what should be reported to the evaluatee and by whom.
  4. determines what the evaluator needs to report to the client
  5. chooses guidelines to follow to implement the evaluation

2.2 Design the methods used for the evaluation

This step is skipped if an already existing evaluation tool is used.

The evaluator alone( or the evaluator with the client):

  1. 根据2.1.5中的指南选择用于评估的标准
  2. 确定需要收集的证据或数据，用以支撑遴选每个标准
  3. 确定可使用的范例
  4. 确定收集数据或证据的方法
  5. 制定收集数据的方案
  1. chooses criteria to use for the evaluation based on the guidelines in Step 2.1.5
  2. determines the evidence that will be collected for each chosen criterion.
  3. determines the sample that will be used, if appropriate.
  4. determines ways to collect the evidence
  5. sets up a plan to collect the evidence.

2.3 Set standards and collect evidence

  1. 评估者告知客户用于评估质量的尺度
  2. 客户基于评估的性能指标制定决策过程
  3. 客户制定用于决策的标准

After the client has set the standards, the evaluator:

  1. 收集数据
  2. 将结果整理成文档
  1. The evaluator informs the client of the scales that will be used to determine quality.
  2. The client develops the decision-making process to use based on the evaluated performance quality.
  3. The client sets the standards that will be used in decision-making.

After the client has set the standards, the evaluator:

  1. collects the evidence.
  2. documents the findings.

2.4 Report and make decisions

  1. 评估者向客户汇报2.1.4中确定的汇报内容
  2. 客户对照2.3.3中的标准检查汇报内容质量
  3. 客户基于结果制定和实施决策
  4. 客户和/或评估者将结果适当地文档化
  5. 如果合适，要么评估者要么客户将评估结果汇报给被评估者
  1. The evaluator reports the information in 2.1.4 to the client.
  2. The client checks the quality against the standards set in 2.3.3.
  3. The client makes and implements decisions based on the findings.
  4. The client and/or evaluator documents the results appropriately.
  5. Either the evaluator or client reports the findings to the evaluatee, if appropriate.

3. Discussion

3.1 Define the parameters of the evaluation

  1. The first step in setting up an evaluation is to determine the need for the evaluation. If there is no need to collect evidence and no decisions will be made based on the evidence, there is a little point in an evaluation. Having the purpose in place gives a framework to design process.
  2. Before designing an evaluation process, the client must decide how the results will be used and what decisions need to be made. Knowing these requirements in advance facilitates finding reliable criteria and determining the evidence needed.
  3. Using the rationale for the evaluation, the client can determine what results, if any, need to be reported to the evaluatee. Often during indirect evaluations, in which the level of one group’s performance is used to evaluate or assess another groups’ performance, no feedback is given to the evaluatee. On the other hand, when decisions are being made that affect the evaluatee directly, a plan should be in place before the outcome is known for reporting the information to the evaluatee.
  4. The client will make decisions based on the evaluation, so the client must know what evidence he or she needs to make a decision. This evidence might be an average, an individual score, or an annotated report. This information must be provided to the client by the evaluator.
  5. Before planning the methods of evaluation, it is important to sketch out the time for completion, checks for reliability, and lists of needed and not acceptable quality criteria. Developing these guidelines ahead of time ensures that the evaluation will align with the client’s needs.

3.2 Design the methods used for the evaluation.

  1. The parameters set in 2.1 should be used to help determine the criteria used in the evaluation process.
  2. Once the criteria are set, the evaluator determines what evidence should be collected. The time for collection, the cost, and the usefulness of the evidence must be considered in deciding what evidence to collect.
  3. Particularly in indirect evaluation, a sample of performance is often used to determine quality. This sample must be unbiased and large enough for the results to be useful.
  4. Determining how to collect the data includes deciding what form of evidence will be used and how it will be collected.
  5. The evidence collection plan includes how, when, and where the information will be collected.

3.3 Set standards and collect evidence from a performance or outcome

  1. Once it is determined what evidence will be collected, a scale must be set to describe how the quality is judged. This scale could be a number scale, a rubric, or a description.
  2. For the process to be unbiased, the client must decide before receiving the information from the evaluation what consequences will occur based on the outcome of the evaluation. This would be a simple process, such as for high quality, there will be positive consequences; for low quality, there will be negative consequences; and between the two, other evaluative methods will be used to determine the consequences.
  3. Once the scale and the decisions to be made are known, the client can set standards for decisions. These standards define “quality” based on what will be reported by the evaluator.
  4. Following the plan in 3.2, the evaluator collects the needed information and analyzes it.
  5. The evaluator documents what has been found in the form of individual scores, level of performance, averages, or narratives. Results should be documented in a way that helps the evaluator write the requested report to the client, using the scales that informed the setting of standards.

3.4 Report and make decisions

  1. For criteria-based standards, after the client has set the standards, the evaluator can give the report to the client, and the standards will not be biased by the evaluative outcome. For norm-based standards, the evaluator must include in the report summaries of outcomes of all evaluations.
  2. Once the client has set the standards and has received the evaluator’s report, the client can check the evaluative outcomes against the standards that have been set.
  3. Based on 3.4.1 and evaluative outcomes, the client makes the decision and develops plans to implement it.
  4. Anything that needs to be documented for future use should be done at this point.
  5. The contents of a report to the evaluatee, if appropriate, should be guided by the results of 4.1.5
